What is jQuery Mobile.

  • Html5 Mobile User Interface
  • Harnesses the power of jQuery
  • Works on many devices

jQuery Mobile phone support.

  1. A-grade: Full enhanced experience with Ajax-based animated page transitions.

    Apple iOS 3.2-5.0, Android, Windows Phone 7-7.5, Blackberry 7

  2. B-grade: Enhanced experience except without Ajax navigation features.

    Blackberry 5.0, Opera Mini (5.0-6.5), Nokia Symbian^3

  3. C-grade: Basic, non-enhanced HTML experience that is still functional

    Blackberry 4.x, WinMo 5.2

When to use it.

  • Mobile web applications
  • Prototyping
  • Combines well with Phonegap/Cordova
